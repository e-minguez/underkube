I've gathered a fresh batch of links to kick off the new year, ranging from Kubernetes networking and BSD tools to security talks from the 39C3 congress.

## Cloud, BSD & Infrastructure

* [**Kubernetes Egress Control with Squid**](https://interlaye.red/kubernetes_002degress_002dsquid.html) / [**Hacker News Discussion**](https://news.ycombinator.com/item?id=46419659): A guide on using Squid proxy to control egress traffic in Kubernetes clusters.
* [**BSD Tools Collection**](https://mfsbsd.vx.sk/): A collection of BSD utilities including [**mfsBSD & mfslinux**](https://mfsbsd.vx.sk/) (minimal memory-based distributions), [**miniBSD**](https://www.minibsd.org/index.php) (small footprint FreeBSD), and [**occambsd**](https://github.com/michaeldexter/occambsd) (an application of Occam's razor to FreeBSD).
* [**Blackship**](https://github.com/seuros/blackship) - A FreeBSD jail orchestrator with TOML config, ZFS, VNET, and Jailfile templates.
* [**Community Cafes**](https://wiki.bsd.cafe/): Community hubs for operating systems enthusiasts, including the [**BSD Cafe**](https://wiki.bsd.cafe/) and [**illumos Cafe**](https://illumos.cafe/#about).
* [**Building Container Images on SLE BCI**](https://opensource.suse.com/bci-docs/guides/building-on-top-of-bci/) - Documentation on building container images based on SUSE Linux Enterprise Base Container Images.
* [**Adafruit nRF52 Bootloader**](https://github.com/oltaco/Adafruit_nRF52_Bootloader_OTAFIX) - USB-enabled bootloaders for the nRF52 BLE SoC chips.
* [**raspi-config**](https://github.com/RPi-Distro/raspi-config/blob/master/raspi-config) - The source code for the Raspberry Pi configuration tool.

## DevOps & Tools

* [**regclient**](https://github.com/regclient/regclient) - A Docker and OCI Registry Client in Go, providing tooling for managing registries.
* [**traefik/whoami**](https://github.com/traefik/whoami) - A tiny Go server that prints OS information and HTTP request details, useful for testing.
* [**macau**](https://github.com/realOrangeOne/macau) - A simple and efficient URL shortener.
* [**Cobalt**](https://cobalt.tools/) - A clean media downloader for various platforms that avoids ads, trackers, and other "nonsense".

## Security & AI

* [**MongoBleed explained simply**](https://bigdata.2minutestreaming.com/p/mongobleed-explained-simply) - An explanation of the MongoBleed vulnerability.
* [**Debunking the AI food delivery hoax**](https://www.platformer.news/fake-uber-eats-whisleblower-hoax-debunked/) - Platformer investigates a viral "whistleblower" story that turned out to be fabricated with AI.
* [**Hacking washing machines (39C3)**](https://media.ccc.de/v/39c3-hacking-washing-machines) - A talk from the 39th Chaos Communication Congress about reverse engineering washing machines.
* [**Brainf*ck: The Ultimate AGI Test**](https://teodordyakov.github.io/brainfuck-agi/) - A discussion on using the esoteric programming language as a benchmark for Artificial General Intelligence.
* [**The AI Engine**](https://sf2platinum.wordpress.com/2017/01/20/the-ai-engine/) - A retrospective look (from 2017) at AI engines, relevant in the context of modern developments.

## SDR & Radio

* [**SigDigger**](https://github.com/BatchDrake/SigDigger) - A Qt-based digital signal analyzer using the Suscan core and Sigutils DSP library.
* [**PySDR**](https://pysdr.org/content/intro.html) - A comprehensive guide to Software Defined Radio and Digital Signal Processing using Python.
* [**PiFmRds**](https://github.com/ChristopheJacquet/PiFmRds) - An FM-RDS transmitter using the Raspberry Pi's PWM output.
* [**BattyBirdNET-Pi**](https://github.com/rdz-oss/BattyBirdNET-Pi) - A realtime acoustic bat and bird classification system for the Raspberry Pi 4/5.

## Linux, Customization & Web

* [**Ambxst**](https://axeni.de/ambxst/) / [**Quickshell Tutorial**](https://www.tonybtw.com/tutorial/quickshell/): Ambxst is a customizable Wayland shell built with [**Quickshell**](https://www.tonybtw.com/tutorial/quickshell/), a framework for creating Qt/QML-based desktop shells.
* [**Asahi Linux (39C3)**](https://media.ccc.de/v/39c3-asahi-linux-porting-linux-to-apple-silicon) - An update on the state of porting Linux to Apple Silicon devices.
* [**You can make up HTML tags**](https://maurycyz.com/misc/make-up-tags/) - A blog post exploring the flexibility of HTML and custom tags.
* [**Glamorous Christmas**](https://marcoroth.dev/posts/glamorous-christmas) - A post about bringing "Charm" to the Ruby programming language.

## General & Misc

* [**39C3: Power Cycles**](https://media.ccc.de/b/congress/2025) / [**Design for 3D-Printing**](https://media.ccc.de/v/39c3-design-for-3d-printing): Talks from the recent Chaos Communication Congress in Hamburg.
* [**Essentials for Pixels**](https://github.com/sameerasw/essentials) - Essential tools, mods, and workarounds for Google Pixel phones and other Android devices.
* [**Was Daft Punk Having a Laugh?**](https://www.madebywindmill.com/tempi/blog/hbfs-bpm/) - An analysis of the tempo choices in Daft Punk's "Harder, Better, Faster, Stronger".
* [**Vitaki-Vitarps5**](https://github.com/mauricio-gg/vitaki-vitarps5) - A fork of the Chiaki PS Vita port for remote play.
* [**What an unprocessed photo looks like**](https://maurycyz.com/misc/raw_photo/) - A visual comparison and explanation of RAW vs processed photography.
