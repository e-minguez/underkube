I missed last week's post because I was too busy trying to beat my high score in UEFI Zaxxon. To make it up to you, here is a double-sized roundup covering the last two weeks.

## Cloud & Kubernetes

* [**Kubernetes Dashboard Archiving**](https://groups.google.com/g/kubernetes-sig-ui/c/vpYIRDMysek/m/wd2iedUKDwAJ?utm_medium=email&utm_source=footer) - A discussion in the Kubernetes SIG-UI group regarding the potential archiving of the Kubernetes Dashboard project.
* [**Ingress Controllers Spreadsheet**](https://docs.google.com/spreadsheets/d/191WWNpjJ2za6-nbG4ZoUMXMpUK8KlCIosvQB0f-oq3k/view?gid=907731238#gid=907731238) - A comprehensive, community-maintained spreadsheet comparing various Kubernetes Ingress Controllers.
* [**MinIO Alternatives**](https://rmoff.net/2026/01/14/alternatives-to-minio-for-single-node-local-s3/) - A look at alternatives to MinIO for single-node local S3 compatible storage, such as SeaweedFS and Garage.
* [**Copacetic**](https://project-copacetic.github.io/copacetic/website/) - A CLI tool for directly patching container images using reports from vulnerability scanners.
* [**KubeVirt Stretched L2**](https://kubevirt.io/2025/Stretched-layer2-network-between-clusters.html) - A guide on setting up a stretched Layer 2 network between clusters using KubeVirt.
* [**ORAS Quickstart**](https://oras.land/docs/quickstart) - Quickstart guide for ORAS (OCI Registry As Storage), allowing you to store arbitrary artifacts in OCI registries.

## DevOps, AI & Tools

* [**SkyVM Announcement**](https://x.com/i/status/2012231037850644961) - Dioxus Labs announces SkyVM, a platform for instant-boot cloud virtual machines designed for AI agent coding tasks.
* [**Gas Town**](https://steve-yegge.medium.com/welcome-to-gas-town-4f25ee16dd04) / [**GitHub Repo**](https://github.com/steveyegge/gastown): Steve Yegge's latest project, "Gas Town", a new take on the IDE for 2026 designed to orchestrate multiple AI agents.
* [**The Key to Agentic Success**](https://thenewstack.io/the-key-to-agentic-success-let-unix-bash-lead-the-way/) - An article arguing that the Unix philosophy and Bash are the keys to success for AI agents.
* [**Bash Course**](https://github.com/bahamas10/bash-course) - A repository containing a course on Bash scripting.
* [**Mosh Static Multiarch**](https://github.com/blinksh/mosh-static-multiarch) - Static binaries for Mosh (Mobile Shell) supporting multiple architectures.
* [**OpenCode**](https://opencode.ai/) - A platform for open AI coding.

## Linux, System & Hardware

* [**Win8DE**](https://github.com/er-bharat/Win8DE) - A project that recreates the Windows 8 Desktop Environment experience on Linux using Wayland.
* [**UEFI Zaxxon**](https://www.tomshardware.com/software/programming/developer-recreates-classic-shoot-em-up-zaxxon-as-a-uefi-firmware-isometric-arcade-game-coded-in-x86-assembly-for-no-os-represents-total-freedom-from-big-tech) / [**Firmware Security Tag**](https://firmwaresecurity.com/tag/games/) / [**GitHub Repo**](https://github.com/mycroftsnm/UEFIGame): A developer has recreated the classic arcade game Zaxxon to run directly as UEFI firmware, written in x86 assembly.
* [**UAPI Configuration Files**](https://uapi-group.org/specifications/specs/configuration_files_specification/) - The Userspace API (UAPI) Group's specification for configuration files in Linux systems.
* [**Kernel Bugs**](https://pebblebed.com/blog/kernel-bugs) - A blog post from Pebblebed discussing Linux kernel bugs.
* [**ESPTimeCast**](https://github.com/mfactory-osaka/ESPTimeCast) - A WiFi-connected LED matrix clock and weather station based on ESP8266/ESP32.

## Web & Design

* [**What happened to WebAssembly?**](https://emnudge.dev/blog/what-happened-to-webassembly/) - A blog post exploring the current state and adoption of WebAssembly.
* [**Amiga Pointer Archive**](https://heckmeck.de/pointers/) - A museum-like collection of mouse pointers from the Amiga era.
* [**C.A.F.E**](https://fezvrasta.github.io/cafe/) - The "Third Way" for Home Assistant automations. Node-RED-style visual editing that compiles to 100% native YAML. No external engine required.
