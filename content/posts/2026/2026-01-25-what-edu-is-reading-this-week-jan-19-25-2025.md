Here is the roundup of links for this week:

## Cloud & Infrastructure

* [**Exe.dev**](https://exe.dev/) - A service to spin up instant, publicly accessible virtual machines with persistent disks.
* [**Zarf**](https://zarf.dev/) - An open-source tool for declarative software delivery into air-gapped and constrained environments.
* [**K3s + Sysbox**](https://docs.k3s.io/blog/2025/09/27/k3s-sysbox) - A guide on using the Sysbox container runtime with K3s.
* [**Cloud Native RAG**](https://medium.com/h7w/cloud-native-rag-embeddings-vectors-fully-self-hosted-82d57142353e) - A walkthrough on building a fully self-hosted Cloud Native RAG stack with embeddings and vectors.
* [**Traefik Middleware Manager v4.1.2**](https://www.reddit.com/r/PangolinReverseProxy/s/U73IbSyl6P) - Now works independent with Traefik and pangolin with native mTLS support per resource.
* [**Ansible OpenWRT Framework**](https://github.com/DigitecGalaxus/ansible-openwrt-framework) - A framework for managing OpenWRT devices at scale using Ansible.
* [**Rockstor**](https://rockstor.com/) - An advanced, open-source NAS operating system built on openSUSE and Btrfs.
