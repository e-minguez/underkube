AI, LLMs & Automation üîó

  • Model Context Protocol (MCP) Introduction: Documentation for the Model Context Protocol, an open standard designed to enable AI agents (LLMs) to interact securely and predictably with external tools, APIs, and data.

  • Jules: The Autonomous Agent CLI Extension / Jules in 90 seconds / Jules GitHub Repo: The official page (and a GitHub repository for a Gemini extension) of Jules, an asynchronous coding sidekick, letting developers delegate long-running tasks.

  • openSUSE systemd-mcp: The openSUSE project implementing an MCP server to allow AI assistants to interact with and manage Linux systems via systemd (e.g., start/stop systemd services).

  • MCP Host CLI (mark3labs/mcphost): A CLI host application that manages connections between LLMs (like Claude, Gemini, Ollama) and external tools through the Model Context Protocol.

  • Turning Gemini CLI into a Multi-Agent System: A guide detailing how to transform the Gemini CLI into a multi-agent orchestration system using clever prompting techniques alone.

  • Gemini CLI Tips and Tricks: A helpful repository collecting best practices and custom command examples for using the Gemini command-line interface efficiently.

  • CERN’s General Principles for AI Use: The official ethical and operational guidelines set by CERN for the use of artificial intelligence within the organization.

Operating Systems & Linux Kernel üîó

  • FreeBSD 15.0 Release Notes: The official release notes for the new major version of FreeBSD.

  • KDE Goes All-In on Wayland: KDE’s official blog post reaffirming their commitment to making the Wayland protocol the primary and default display server for the Plasma desktop environment.

  • ZFS-Enabled Disaster Recovery Virtualization: A guide on leveraging ZFS snapshots and replication for efficient disaster recovery and live migration of virtual machines.

  • Building an IPv6-Focused OpenBSD Home Router: A technical guide on setting up a robust, security-focused home network gateway using OpenBSD with an emphasis on IPv6 configuration.

  • Macbook 12’ (2015, 2016, 2017) Omarchy Linux Installation Notes: A Gist containing configuration and instructions to improve certain MacBook models running Omarchy.

  • GNOME Shell Extension: Activate GNOME: A minimal GNOME Shell extension that allows users to mimic one of the best Windows 11 features, the activation watermark!.

  • Aluminium OS: Android for PCs: An article discussing Aluminium OS the alleged new Google’s operating system to replace Chrome OS.

Cloud Native & DevOps üîó

  • kubeswitch: A powerful command-line tool for quickly switching between different Kubernetes cluster contexts using fuzzy finding.

  • git-remote-s3: An AWS Labs project that enables using Amazon S3 as a Git remote and LFS server, providing a serverless option for hosting repositories.

  • Canonical Chisel / Red Hat Hummingbird: Ultra-minimal container images from Canonical and Red Hat.

  • Penpot: The open-source design and prototyping tool for design and code collaboration (an alternative to Figma/Sketch).

  • RESTerm: A terminal-based client for working with HTTP, GraphQL, gRPC, and WebSockets/SSE, built to be keyboard-first and offline-friendly.

  • Disk Burn-In and Testing Script: A collection of scripts for stress-testing new hard drives and SSDs to catch early failures (burn-in testing).

Programming & Learning üîó

  • One OS Thread, Millions of Goroutines: A detailed explanation of the Go runtime scheduler, detailing how it manages concurrency efficiently using the M:N scheduler model.

  • Linux Kernel Explorer: An interactive online tool that allows users to browse and study the Linux kernel source code with rich cross-referencing capabilities.

  • Wails: A framework for building cross-platform desktop applications using Go (backend logic) and standard web technologies (frontend UI).

  • Binary Exploitation Guide: A comprehensive, structured guide intended for newcomers wanting to learn the fundamentals of binary exploitation and reverse engineering.

  • We stopped roadmap work for a week and fixed 189 bugs: An article on the benefit of allocating time technical cleanups and fixes, improving both code quality and developer morale.

Security & Privacy üîó

  • Exposed By Default: A scary demonstration showing exactly how much data (fingerprints, hardware info, network data) your web browser gives away automatically on every page load.

  • If OpenSSL Were a GUI: A blog post visualizing how the complex commands and processes of using OpenSSL for PKI (Public Key Infrastructure) might look in a modern graphical interface.

  • Hacking the Agent (HackTheAgent.com): LLM Security Challenge.

  • Google Antigravity Exfiltrates Data: Security research detailing a data exfiltration vulnerability found in Google’s Antigravity AI development platform.

  • Chameleon Ultra: A powerful, open-source pocket-sized device for reading, emulating, and cloning various NFC and RFID tags.

Culture, Retro & Fun üîó

  • 11 Years Later and I Still Think About This Ad: A deeply sad and memorable viral advertisement from over a decade ago.

  • Kidoom: KiDoom: Running DOOM on PCB Traces.

  • Cool Retro Term: A graphical terminal emulator that intentionally mimics the look and feel of old cathode ray tube monitors.

  • Bloodborne on Steam Deck is More Playable Than Ever: News on significant performance improvements to the Shadow PS4 emulator, making the notorious PlayStation exclusive Bloodborne highly playable on the Steam Deck.

  • Cookie Consent Speed Run: Funny small game. Try to access the website without acceptig cookies.

  • introducing copyparty, the FOSS file server: All protocols, all features, single project. Impressive.

  • How to gracefully collapse a building: A demonstration of the science and engineering behind controlled building demolitions.
